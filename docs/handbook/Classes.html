Classes

  • Promise

    A promise represents the future value of a (usually) asynchronous task.

    To obtain the value of a promise we call then.

    Promises are chainable: then returns a promise, you can call then on that promise, which returns a promise, you can call then on that promise, et cetera.

    Promises start in a pending state and resolve with a value to become fulfilled or an ErrorProtocol to become rejected.
